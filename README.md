##🧠 **Machine Learning: Práticas e Conceitos (2026.1)**
Este repositório foi criado para documentar minha jornada na disciplina optativa de **Aprendizado de Máquina** do curso de **Tecnologia em Análise e Desenvolvimento de Sistemas (TADS)** na **UFRN** durante o semestre 2026.1.

O foco aqui é a aplicação prática de algoritmos de **Machine Learning**, desde o pré-processamento de dados até a construção de redes neurais profundas.
---
#🛠️ **Ferramentas Principais**
* Linguagem: Python 3.x
* Ambiente: Google Colab (GPU acelerada por hardware)
* Frameworks: Keras & TensorFlow
* Manipulação de Dados: Pandas, NumPy e Scikit-Learn
* Visualização: Matplotlib e Seaborn
---
#📚 **Tópicos Estudados**
Ao longo do semestre, o repositório será atualizado com os seguintes temas:
1. Pré-processamento de Dados: Limpeza, normalização e tratamento de outliers.
2. Aprendizado Supervisionado: Regressão Linear, Árvores de Decisão e KNN.
3. Redes Neurais (Keras): Perceptrons, Redes Multicamadas (MLP) e funções de ativação.
4. Otimização: Estudo de funções de perda (loss) e otimizadores (Adam, SGD).
5. Visão Computacional: Classificação de imagens com CNNs.
---
#🚀 **Como Executar as Práticas**
Todos os notebooks estão configurados para rodar diretamente no **Google Colab**. Para visualizar ou testar:
1. Acesse a pasta correspondente ao exercício.
2. Abra o arquivo ```.ipynb``` desejado.
3. Execute as celulas para reproduzir os treinamentos (Faça o download e upload no ambiente do Colab).
